How does concertian work?

Core of all this is concertian- planning tool for organizers.

The organizer will check the date and create an event (concert). All this simply by using concertian system for managers.

www.manager.concertian.com

Then, all the information- time, place and performers are given to you. Any changes and new concerts will be displayed in real time when being done by the organizer.
